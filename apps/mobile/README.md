# Clearity Mobile (iOS)

iOS app built with Capacitor. Wraps the Vercel-hosted web app in a native webview.

## Prerequisites

- Xcode (latest version)
- Apple ID signed in to Xcode (Signing & Capabilities > Team)
- CocoaPods (`sudo gem install cocoapods` if not installed)

## Development

### First-time setup

```bash
cd apps/mobile
npx cap add ios        # Already done
npx cap sync           # Sync web assets & plugins
npx cap open ios       # Open in Xcode
```

### Run on Simulator

1. `npx cap open ios`
2. Select a simulator in Xcode
3. Press `Cmd+R` to build and run

### Run on Physical Device

1. Connect device via USB
2. Select the device in Xcode
3. Trust the developer profile on the device (Settings > General > Device Management)
4. Press `Cmd+R`

## Build for Distribution

### TestFlight (Recommended for beta testing)

1. Open Xcode: `npx cap open ios`
2. Select **Product > Archive**
3. In Organizer, click **Distribute App > App Store Connect**
4. Upload to App Store Connect
5. In [App Store Connect](https://appstoreconnect.apple.com), add testers to TestFlight

### App Store

1. Requires Apple Developer Program ($99/year)
2. Archive and upload via Xcode (same as TestFlight)
3. Submit for review in App Store Connect

## Configuration

- **Bundle ID:** `io.clearity.app`
- **Server URL:** `https://clearity-taupe.vercel.app` (configured in `capacitor.config.ts`)

## Notes

- The app loads the Vercel-hosted web app directly — no local build required
- `ios/` directory is generated by Capacitor and can be regenerated with `npx cap add ios`
- After changing `capacitor.config.ts`, run `npx cap sync` to apply changes
